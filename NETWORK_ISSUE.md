# 网络限制问题说明

## 当前状态
✅ API配置正确
✅ 环境变量已设置
❌ 网络连接受限

## 问题原因
当前沙箱环境无法直接访问 Polymarket API（网络超时）。

这是由于云环境的网络限制导致的，**不是配置问题**。

## 解决方案

### 方案1：使用模拟模式（当前可用）
系统已支持模拟市场数据和模拟交易，可以：
- 测试完整的交易流程
- 验证策略逻辑
- 熟悉系统操作

### 方案2：部署到支持网络的环境
将项目部署到：
- Vercel
- Railway
- AWS
- Google Cloud
- 您自己的服务器

### 方案3：本地运行
如果您有本地开发环境：
```bash
# 1. 克隆项目
git clone <your-repo>

# 2. 复制配置
cp .env.local ./

# 3. 安装依赖
pnpm install

# 4. 启动服务
pnpm dev

# 5. 访问
http://localhost:5000
```
