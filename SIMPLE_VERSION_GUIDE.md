## 📁 文件清单

只需要创建以下 5 个文件：

### 1. package.json
### 2. index.html
### 3. app.js
### 4. styles.css
### 5. .env

---

## 第一步：创建项目目录

在 Windows 上创建文件夹：
```
C:\polymarket-simple
```

## 第二步：创建 package.json

创建文件 `package.json`：

```json
{
  "name": "polymarket-simple",
  "version": "1.0.0",
  "scripts": {
    "start": "npx serve -l 5000"
```
